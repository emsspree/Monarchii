Monarch
=======
*Monarch, a theme for Sublime Text.*

Use VS instead of Sublime? [Check out the VS port of Monarch](https://github.com/MiloradFilipovic/monarch-vs-code) courtesy of [@MiloradFilipovic](https://github.com/MiloradFilipovic) (Thanks! üçª)

## Install

Find and install the *Monarch* package using [Package Control](https://sublime.wbond.net), or manually clone this repo into your Sublime Text's `Packages` folder.
